🏎️ RaceGame – Консольная гоночная игра на 7 языках
Мчись по трассе, уклоняйся от препятствий и устанавливай рекорды!
Простая, но захватывающая аркадная гонка в терминале.
Управляй машинкой с помощью клавиш, избегай столкновений и набирай очки.
Реализована на 7 языках программирования с единым интерфейсом и цветным выводом.

🚀 Возможности
Динамичная трасса – препятствия появляются случайно и движутся сверху вниз.

Управление – стрелки влево/вправо (или A/D) для перемещения машинки.

Счёт – очки начисляются за каждый успешно пропущенный ряд препятствий.

Ускорение – игра постепенно ускоряется, повышая сложность.

Рекорды – лучший результат сохраняется в локальный файл.

Цветной вывод – машинка, препятствия и дорога выделены цветами.

Интерактивность – игра заканчивается при столкновении, показывает счёт и рекорд.

Кроссплатформенность – работает в терминале Windows, Linux и macOS.

📖 Управление
Стрелка влево или A – движение влево.

Стрелка вправо или D – движение вправо.

Пробел – пауза (в некоторых реализациях).

Q – выход из игры.

🛠 Установка и запуск
Python
bash
python race.py
Требуется Python 3.6+.

C++
bash
g++ -std=c++17 race.cpp -o race
./race
Требуется компилятор с поддержкой C++17 и библиотека pthread (обычно встроена).

Go
bash
go build race.go
./race
Требуется Go 1.16+. Используется пакет golang.org/x/term (автоматически загружается).

JavaScript (Node.js)
bash
node race.js
Требуется Node.js 12+.

C#
bash
csc race.cs
mono race.exe   # или dotnet run
Требуется .NET Framework или .NET Core.

Ruby
bash
ruby race.rb
Требуется Ruby 2.5+.

Java
bash
javac race.java
java race
Требуется Java 11+.

🧠 Логика игры
Игровое поле – дорога шириной 7 клеток, игрок находится в нижнем ряду.

Препятствия – случайные символы появляются в верхней части и с каждым кадром опускаются на одну позицию вниз.

Управление – игрок может перемещаться влево или вправо на одну позицию за ход.

Столкновение – если препятствие достигает позиции игрока, игра заканчивается.

Очки – за каждый ряд, успешно пропущенный без столкновения, начисляется 1 очко.

Скорость – с увеличением счёта игра ускоряется (задержка между кадрами уменьшается).

✨ Дополнительные фичи
Анимация – плавное движение с помощью задержек.

Цвета – машинка (зелёная), препятствия (красные), дорога (серая).

Рекорд – сохраняется в файл ~/.race_record (или аналогичный).

Пауза – нажатие P или Space приостанавливает игру (в некоторых реализациях).

Завершение – после игры показывается счёт и рекорд, предлагается начать заново.
